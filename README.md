# CALCUMIN

Calculadora minera basada en Vite + React + TypeScript.
Esta versión está adaptada para despliegue público (sin login/token obligatorio).

## Configuración para GitHub Pages
Para publicar en `flavio1227/CALCUMIN-Public`, usa:
```
VITE_BASE_PATH=/CALCUMIN-Public/
```

## Desarrollo
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Publicar en GitHub
1. Crear repo: `flavio1227/CALCUMIN-Public`
2. Subir este proyecto al repo.
3. Configurar GitHub Pages para servir desde la rama principal (o `gh-pages`, según tu flujo).
